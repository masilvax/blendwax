Import from Discogs

  • This application uses Discogs’ API but is not affiliated with, sponsored or endorsed by Discogs. ‘Discogs’ is a trademark of Zink Media, LLC.
  • Press the START button to import or update your collection from Discogs.
  • Due to Discogs’ request limits, only one release per second can be imported.
  • Only those releases that are not present in BlendWax will be imported. If the release has been already imported to BlendWax by some other user, there's no need to send request to Discogs. The release will simply be added to your collection.
  • You can stop the import at any moment and run it again later. Releases not imported during this import session will be imported in the next session.
  • Import and update in BlendWax works only for vinyl releases.
